# SVGEditBench
This repository will contain the dataset that was used for the paper "SVGEditBench: A Benchmark Dataset for Quantitative Assessment of LLM's SVG Editing Capabilities". The paper was accepted to [Workshop on Graphic Design Understanding and Generation (GDUG)](https://sites.google.com/view/gdug-workshop), a workshop held in the CVPR2024 conference.

The details and the data itself will be comming soon!
